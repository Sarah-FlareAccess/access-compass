IMAGE GUIDELINES
================
- Recommended size: 800x600px or 4:3 aspect ratio
- Format: JPEG or PNG
- Keep file sizes under 200KB for fast loading
- Use realistic photos where possible
- Ensure images have good contrast for visibility
