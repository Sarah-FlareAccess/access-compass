# Gap Analysis: Access Compass vs AS 1428.1:2021 and AS 1428.2:1992

*Generated: 2026-02-11*

## Legislative Context
- **Premises Standards 2010 (amended 2024)** now reference AS 1428.1:2021 (not 2009)
- Only requirements called up by Premises Standards are MANDATORY
- AS 1428.2:1992 is ENHANCED/BEST-PRACTICE (covers 90% of users vs 80% in AS 1428.1)

---

## HIGH PRIORITY GAPS (Mandatory - referenced in Premises Standards)

### 1. Signage illumination (APS D4.5)
- Signs must be illuminated to maintain luminance contrast
- Module: 3.3 (Wayfinding) or 4.2 (Signage)

### 2. Hearing augmentation coverage percentage (APS D3.7)
- 80% for induction loop, 95% for receiver systems
- Module: 3.5 (Hearing Augmentation)

### 3. Hearing augmentation system type on signage (APS D3.6(b)(ii))
- Signage must indicate both the area served AND the type of system
- Module: 3.5 (Hearing Augmentation)

### 4. Receiver availability signage (APS D3.6(b)(iii))
- Where receivers can be obtained must be signed
- Module: 3.5 (Hearing Augmentation)

### 5. Toilet LH/RH configuration signage (APS D3.6(c))
- Signage must indicate left-hand or right-hand transfer
- Module: 3.2 (Toilets)

### 6. Ambulant toilet signage (APS D3.6(d))
- Signage identifying ambulant toilet facilities required
- Module: 3.2 (Toilets)

### 7. Screens supplementing PA systems (APS D3.7(4))
- Screens capable of displaying public announcements must supplement PA
- Module: 3.5 (Hearing Augmentation) or 6.x (Events)

### 8. Raised tactile information height band (APS D3.6(b))
- Raised tactile information between 1200-1600mm
- Module: 4.2 (Signage)

### 9. Title Case on signage (APS D4.3(2))
- Sign text must be in Title Case
- Module: 4.2 (Signage)

### 10. Number of accessible parking bays (AS/NZS 2890.6 Table 1)
- Specific ratio of accessible bays to total parking required
- Module: 2.1 (Parking)

### 11. Signage at non-accessible entrances (APS D3.6(e))
- Must direct people to the accessible entrance
- Module: 2.2 (Entrances)

## MEDIUM PRIORITY GAPS (Best-practice, high impact)

### 12. Uniform/adequate lighting assessment (AS 1428.2 Cl. 19)
- Entrances: 150 lx, Passageways: 150 lx, Stairs/ramps: 150 lx
- Toilets: 200 lx, Counters: 250 lx, Displays: 200-300 lx
- Module: 3.4 (Sensory Environment)

### 13. Lighting for lip reading (AS 1428.2 Cl. 19.1 Note 3)
- Minimum 150 lx without glare at service areas
- Module: 3.4 (Sensory Environment)

### 14. Emergency call buttons in toilets (AS 1428.2 Cl. 15.1(c))
- Near WC pan, shower, and bath
- Module: 3.2 (Toilets)

### 15. Visual alarms in enclosed spaces (AS 1428.2 Cl. 18.2)
- Toilets, meeting rooms, sleeping areas where people may not see main alarms
- Module: 3.7 (Emergency Preparedness)

### 16. General controls operability (AS 1428.2 Cl. 23)
- Light switches, power outlets, thermostats: one hand, no grasping/pinching
- Module: 3.1 (Paths & Aisles)

### 17. Acoustic design elements (Checklist 9.10)
- Noise-reducing features: rubber tips, soft furnishings, carpet, double glazing
- Module: 3.4 (Sensory Environment)

### 18. Vertical clearance in car parks (AS 1428.2 Cl. 14.2)
- Minimum 2500mm for wheelchair roof hoists
- Module: 2.1 (Parking)

### 19. Accessible podium/stage access (AS 1428.2 Cl. 26.2)
- Ramped access, 360-degree turn space, controls from seated position
- Module: 3.6 or 6.x (Events)

### 20. Vending machine accessibility (AS 1428.2 Cl. 29)
- Controls 500-1200mm, one-hand operation, tactile, wheelchair circulation
- Module: 3.1 or 3.6

### 21. Drinking fountain/water cooler (AS 1428.2 Cl. 27.3)
- Wheelchair approach, controls, dimensions
- Module: 3.1 or 3.4

### 22. Door handle colour contrast (AS 1428.2 Cl. 23.3(b))
- 30% luminance contrast with background
- Module: 3.1 (Internal Doors)

## ENHANCEMENTS TO EXISTING QUESTIONS

### E1: Parking (Module 2.1)
- Add bay count relative to total parking (MANDATORY)
- Add bay dimension assessment (MANDATORY)
- Add vertical clearance for covered car parks (best-practice)

### E2: Hearing augmentation signage (Module 3.5)
- Distinguish "signage indicates area" vs "signage identifies system type"
- Add "signage shows where to get receivers" option

### E3: Accessible toilet (Module 3.2)
- Add LH/RH configuration signage (MANDATORY)
- Add emergency call button (best-practice)

### E4: Internal doors (Module 3.1)
- Add door handle contrast option
- Add maximum opening force reference (20N)
- Add door closer assessment

### E5: Glass door contrast (Module 2.2/3.1)
- Add specific dimensions: 75mm wide, bottom at 900-1000mm

### E6: Stair nosings (Module 2.2)
- Add dimensional guidance in question: 50mm tread, 30mm riser
- Add slip-resistance assessment

### E7: Lighting (Module 3.4)
- Add baseline uniform/adequate lighting question BEFORE adjustability
- Add glare-free assessment

### E8: Counters/tables (Module 3.6)
- Add specific dimensions: 850mm +/-20mm, 820mm clearance
- Add minimum 900mm accessible counter length

### E9: Ramps (Module 2.2)
- Add minimum unobstructed width (1000mm mandatory, 1200mm enhanced)
- Add landing length requirements (1200mm mandatory)
- Add max ramp length between landings (9m)

### E10: Emergency evacuation (Module 3.7)
- Add visual alarms in enclosed spaces question
- Add alarm audibility vs background noise
