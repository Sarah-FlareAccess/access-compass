# Access Compass - Pricing Options (March 2026)

Working document covering pricing structures explored. See final recommendation at the end.

---

## Key Considerations

- The core product value (assessment + report) is worth $600-900 and must not be undercut by a cheap monthly subscription
- Discovery recommends modules based on touchpoints, so buyers should NOT cherry-pick modules they don't understand
- Multi-site businesses should pay proportionally more, without an honesty-based size declaration
- Councils and enterprises expect a sales conversation and should pay to value
- Resource Hub access gives lasting value beyond a surface-level checklist
- DIAP management, Training Hub, re-assessments, and team access are ongoing value, separate from the assessment itself

### Product Components

| Component | Description |
|---|---|
| Discovery | Guided process that recommends relevant modules based on touchpoints and industry |
| Pulse Check | Summary-level assessment of recommended modules |
| Deep Dive | Full question-by-question assessment with detailed findings |
| Report | Prioritised actions with timeframes (basic for Pulse, detailed for Deep Dive) |
| Resource Hub | Guides, checklists, standards references organised by category |
| Training Hub | Courses and resources (flagship: AI for accessible resources) |
| DIAP Management | Action tracking, categories, change detection, filtering, export |
| Comparison Runs | Re-assess and compare progress over time |

## Option A: Three Tiers + Enterprise (Simplest)

| | Pulse Check | Professional | Comprehensive | Enterprise |
|---|---|---|---|---|
| Price | $600 one-off | $150/mo or $1,500/yr | $300/mo or $3,000/yr | Custom ($5k-15k/yr) |
| Assessment depth | Pulse (summary) | Deep Dive | Deep Dive | Deep Dive |
| Modules | Up to 10 (discovery-recommended) | Up to 20 | All 37 | All 37 + custom |
| Resource Hub | 3 months access | Included | Included | Included |
| Training Hub | No | Limited | Full | Full |
| DIAP Management | No | No | Yes | Yes |
| Report | Basic summary | Full report | Full report + comparison runs | Full + multi-site |
| Users | 1 | 1-3 | 1-10 | Unlimited / departments |

Upgrade path: Pulse Check spend credited toward Professional/Comprehensive within 6 months.

**Pros**: Easy to understand. Clear good/better/best ladder. Discovery engine naturally recommends the right tier.

**Cons**: Rigid. Edge cases (small business wanting DIAP, multi-site businesses unclear where they fit). Enterprise tier tries to cover both "big single venue" and "chain with 40 locations" which are very different needs.

## Option B: Base + Add-Ons (Most Flexible)

### Base Plans

| | Pulse Check | Deep Dive |
|---|---|---|
| Price | $600 one-off | $120/mo or $1,200/yr |
| Depth | Summary-level findings | Full question-by-question |
| Modules | Discovery-recommended set | Discovery-recommended set |
| Report | Basic | Full with priorities and timeframes |
| Resource Hub | 3 months | Included |

### Add-Ons (stack onto Deep Dive)

| Add-on | Price |
|---|---|
| Additional module bundle (5 modules) | +$30/mo |
| All modules unlock | +$80/mo |
| DIAP Management | +$80/mo |
| Training Hub | +$40/mo |
| Additional users (per 3-seat pack) | +$30/mo |

Enterprise/Council bundle: All modules + DIAP + Training + unlimited users = custom pricing ($5k-15k/yr).

DIAP-only option: For orgs that just want DIAP + Resource Hub without assessments = $200/mo.

**Pros**: Businesses pay for exactly what they need. Small cafe pays around $120/mo, large venue around $310/mo, scales naturally. Module bundles less confusing than per-module pricing.

**Cons**: More decisions for the buyer. Add-on fatigue. Needs clear UI to show what's included vs extra. Likely confusing for buyers unfamiliar with the product.

## Option C: Size-Based Tiers (Easiest to Sell)

| | Starter | Growing | Established | Enterprise |
|---|---|---|---|---|
| Target | Sole trader / <5 staff | Small business 5-50 | Medium 50-500 | 500+ / councils / chains |
| Price | $600 one-off | $180/mo or $1,800/yr | $350/mo or $3,500/yr | Custom |
| Assessment | Pulse Check | Deep Dive | Deep Dive | Deep Dive |
| Modules | Discovery-recommended (typically 8-12) | Discovery-recommended (typically 12-20) | All 37 | All 37 + multi-site |
| Resource Hub | 3 months | Yes | Yes | Yes |
| Training | No | Yes | Yes | Yes + custom content |
| DIAP | No | Basic (view/export) | Full management | Full + department views |
| Users | 1 | 1-3 | Up to 10 | Unlimited |

**Pros**: Buyer self-selects in 5 seconds based on size. No module-counting confusion. Price scales with ability to pay.

**Cons**: Relies on honesty system for size declaration. Significant edge cases: tiny business that needs DIAP, large business that only wants a pulse check, small business that wants deep dive. Creates resentment (punished for being bigger). Override/upgrade paths add complexity.

## Option D: Hybrid - Tiers With Module Top-Up

| | Pulse Check | Deep Dive | Full Access |
|---|---|---|---|
| Price | $600 one-off | From $1,200/yr | $3,600/yr |
| Modules included | Discovery set | 12 modules (chosen or discovery-recommended) | All 37 |
| Extra modules | N/A | $120/module/yr | All included |
| Resource Hub | 3 months | Yes | Yes |
| DIAP | No | Add $960/yr | Included |
| Training | No | Add $480/yr | Included |

At around 15 modules + DIAP, the maths naturally pushes toward Full Access ($1,200 + 3x$120 + $960 = $2,520 vs $3,600 for everything). Intentional upgrade trigger.

Enterprise/Council: Full Access + multi-site + unlimited users + onboarding = custom.

**Pros**: Per-module option exists but isn't the default path. Avoids confusing tally. Natural crossover points nudge upgrades.

**Cons**: Slightly more complex than A or C. Risk that buyers dismiss modules they don't understand to save money, missing impactful findings. Discovery recommends similar touchpoints regardless of business size, so module count doesn't naturally distinguish pricing tiers.

## Option E: Two Products + Volume (Assessment Value Protected)

**Rejected variant.** Structured as a $180/mo membership containing the assessment. Problem: businesses could subscribe for one month, get the full assessment and report, then cancel. This undercuts the $800-900 core product value.

## Option F: Assessment Upfront + Optional Membership (Recommended Direction)

Separates the core product (assessment) from the ongoing toolkit (membership) so the assessment value is always captured.

### The Assessment (One-Off Purchase, Always Required)

| | Price |
|---|---|
| Pulse Check (summary-level, all recommended modules) | $600 |
| Deep Dive (full detail, all recommended modules) | $900 |

Both include:
- Full discovery process
- Assessment of ALL discovery-recommended modules (no cherry-picking)
- Prioritised report with timeframes
- Exportable PDF (kept forever)
- Resource Hub access for 3 months

Upgrade path: Pulse Check buyers can upgrade to Deep Dive for $400 (credited difference) within 6 months.

### The Membership (Optional, Ongoing Value-Add)

Only available after purchasing an assessment (except DIAP-only path). Not where the assessment lives. This is the ongoing toolkit.

| | Monthly | Annual |
|---|---|---|
| Membership | $120/mo | $1,200/yr |
| Additional 3-user pack | +$30/mo | +$300/yr |

Membership includes:
- DIAP management system
- Training Hub (full access)
- Resource Hub (ongoing, beyond the 3-month window)
- Re-assessments and comparison runs
- Up to 3 users

Does NOT include the initial assessment. That is always paid separately.

### Multi-Site Pricing

Each site requires its own assessment purchase:

| Additional site | Price |
|---|---|
| Pulse Check per site | $600 |
| Deep Dive per site | $900 |

Membership covers all sites (DIAP, Training, Resource Hub, re-assessments).

### DIAP-Only Path

For orgs that already have audit data and just want the management system:

| | Monthly | Annual |
|---|---|---|
| DIAP + Resource Hub + Training (no assessment) | $140/mo | $1,400/yr |
| Additional sites / users | Same add-on pricing as above | |

### Example Scenarios

| Business | Assessment Cost | Membership | Total Year 1 |
|---|---|---|---|
| Small cafe, just wants to know where they stand | $600 (Pulse) | None | $600 |
| Same cafe, wants to act on it properly | $900 (Deep Dive) | None | $900 |
| Cafe decides they want DIAP and ongoing tools | $900 (Deep Dive) | $120/mo | $2,340 |
| Restaurant group, 4 sites, Deep Dive | 4x $900 ($3,600) | $120/mo | $5,040 |
| Hotel chain, 8 sites | 8x $900 ($7,200) | $120/mo + users | $8,640+ |
| Council, 30 sites | Custom conversation | Custom | $20,000+ |
| Enterprise, DIAP only (no assessment) | None | $140/mo | $1,680 |

### Why This Structure Works

- Assessment value ($600-900) is always captured upfront, no one-month-and-cancel risk
- No module cherry-picking: discovery recommends, the package includes all recommendations
- Multi-site scales naturally: each site pays for its own assessment
- No honesty-based size tiers: pricing is based on number of sites and users, which are concrete
- Membership is genuinely separate value (ongoing tools), not the report repackaged
- Councils and large chains land in custom pricing territory organically (30x $900 = $27k before membership)
- Small businesses are not priced out ($600 Pulse Check is the floor)

### Open Questions

- Is $600/$900 the right Pulse/Deep Dive split?
- Should Pulse Check include fewer modules or just less detail per module?
- What distinguishes Pulse Check findings from Deep Dive findings in the report?
- Should the 3-month Resource Hub window on assessments be longer or shorter?
- At what site count should "contact us for custom pricing" trigger? (Suggested: 10+)
- Should membership include a set number of re-assessments per year, or unlimited?
- Does the DIAP-only path need its own onboarding flow?
