Brand Style Guide — WCAG AA Compliant

✓ WCAG AA Accessibility Compliance

This style guide meets WCAG 2.1 Level AA standards:

  • Color Contrast: All text meets 4.5:1 ratio (7:1 for large text)
  • Focus Indicators: 3px solid outlines with 2px offset on all interactive elements
  • Touch Targets: Minimum 44x44px for all buttons and clickable elements
  • Keyboard Navigation: Full keyboard support with visible focus states
  • Semantic HTML: Proper heading hierarchy and ARIA labels where needed
